ABOUT
MINI STADOOR.

Mini Stadoor is the final project of the ITP Program at ISTAD for the subject Spring Microservices. The project is guided by 2 mentors and built by a team of 9 students as a developer security SaaS prototype.

What Mini Stadoor Is

A developer security SaaS prototype where developers can register an API or frontend, publish dynamic routes, and apply security policies through one platform.

What This Prototype Proves

Dynamic service registration, dynamic routing, Basic Authentication, API Key enforcement, and JWT working together through the Mini Stadoor control plane.

What Comes Next

The wider Mini Stadoor vision includes IAM, OAuth2 with OIDC, BFF as a Service, compliance features, traffic management, and more platform modules.
